# Project Guidelines for Claude

## 블로그 포스팅 규칙

### 예약 발행 규칙
- **하루 1개씩 순차 발행**: 여러 포스트를 한 번에 만들더라도 날짜는 하루씩 증가
- **마지막 포스트 날짜 이후부터 시작**: 새 포스트는 기존 최신 포스트 날짜 + 1일부터
- 특별한 지침이 있을 때까지 이 규칙 유지

### 현재 상태
- 최신 포스트: Episode 17 (2026-01-19)
- 다음 포스트 시작 날짜: 2026-01-20

### 포스트 생성 시 체크리스트
1. `blog/ko/` 에서 최신 포스트 번호 및 날짜 확인
2. 다음 Episode 번호 = 최신 + 1
3. 다음 날짜 = 최신 날짜 + 1일
4. 여러 포스트 생성 시 날짜 순차 증가 (19, 20, 21...)
