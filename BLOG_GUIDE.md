# 블로그 포스팅 가이드

워드프레스처럼 쉽게 SEO 최적화된 블로그 포스트를 작성하는 방법입니다.

## 포스트 작성 형식

```
포커스 키워드: 메인 키워드
관련 키워드: 키워드1, 키워드2, 키워드3
메타디스크립션: 150자 내외의 설명 (검색결과에 표시됨)
날짜: 2025-01-15
이모지: 🚀
색상: indigo-purple

===

# 포스트 제목

본문 내용을 마크다운으로 작성...
```

## 빠른 시작

```bash
# 1. posts/post-{번호}.md 파일 생성 후 위 형식으로 작성

# 2. 빌드 (모든 게 자동!)
npm run build:blog
```

## 메타데이터 필드 설명

| 필드 | 필수 | 설명 |
|------|------|------|
| 포커스 키워드 | O | 이 글의 핵심 키워드 (SEO 최적화) |
| 관련 키워드 | O | 연관 검색어들 (쉼표로 구분) |
| 메타디스크립션 | O | 검색결과에 표시될 설명 (150자 권장) |
| 날짜 | - | 발행일 (없으면 오늘 날짜) |
| 이모지 | - | 카드에 표시될 이모지 (기본: 📝) |
| 색상 | - | 카드 배경색 (기본: indigo-purple) |

## 색상 옵션

| 색상 이름 | 미리보기 |
|-----------|----------|
| `indigo-purple` | 보라색 (기본) |
| `pink-rose` | 핑크색 |
| `amber-orange` | 주황색 |
| `emerald-teal` | 청록색 |
| `blue-cyan` | 파란색 |

| `green-emerald` | 초록색 |

## 마크다운 문법



## 빌드 시 자동 처리 항목

- `blog/post-{번호}.html` 생성
- `blog.html` 포스트 목록 자동 업데이트
- `sitemap.xml` 업데이트
- SEO 메타 태그 생성 (포커스 키워드 + 관련 키워드 병합)
- Open Graph / Twitter Card 태그
- JSON-LD 구조화 데이터
- 소셜 공유 버튼

## 폴더 구조

```
ai-life-summary/
├── posts/              # 마크다운 소스
│   └── post-1.md
├── blog/               # 빌드된 HTML
│   └── post-1.html
├── blog.html           # 목록 페이지 (자동 업데이트)
└── sitemap.xml         # 사이트맵 (자동 업데이트)
```

## SEO 팁

1. **포커스 키워드**는 제목(# 제목)에도 포함하세요
2. **메타디스크립션**은 150자 내외로 작성
3. **첫 문단**에 핵심 내용을 넣으세요
4. **FAQ 섹션**을 추가하면 Featured Snippet에 노출될 확률 증가
5. **이미지**에는 alt 텍스트를 꼭 넣으세요
